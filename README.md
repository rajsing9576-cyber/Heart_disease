Heart Disease Prediction – Project Description
A Heart Disease Prediction System is a Machine Learning project that predicts whether a person is likely to have heart disease based on medical attributes and health conditions.
The system analyzes patient data such as age, blood pressure, cholesterol level, heart rate, chest pain type, and other medical parameters to determine the risk of heart disease.

Objective
The main objective of this project is to:


Predict heart disease at an early stage


Assist doctors in diagnosis


Reduce health risks


Improve medical decision-making using Machine Learning

Working Process
1. Data Collection
The heart disease dataset is collected from medical records.

2. Data Preprocessing


Handling missing values


Converting categorical values


Feature scaling


Splitting dataset into training and testing data



3. Model Training
Machine Learning algorithms are trained using patient data.
Common algorithms:


Logistic Regression


Decision Tree


Random Forest


Support Vector Machine (SVM)



4. Prediction
The trained model predicts whether the patient has heart disease or not.
Output:


0 → No Heart Disease


1 → Heart Disease Present
